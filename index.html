Hey there! 👋 I'm Prem Kumar, and I'm excited to showcase my fun project: the Profile Card! 🖼️ This isn't just any profile card—it's a playful, stylish way to present basic employee information in a sleek format. Crafted using a mix of HTML 📝 and CSS 🎨, this card is designed to blend function with a touch of humor and creativity! ✨


One of the coolest aspects? I used @media screen in CSS to make it responsive across a range of devices 📱💻🖥️. Whether you're viewing it on a smartphone, tablet, or desktop, the design adjusts seamlessly for the best user experience. This project proves that with the right code, simple elements can adapt beautifully across all screens!


And the best part? It's designed with beginners in mind! 🌱 So if you're just starting out in web development, this profile card is a perfect project to flex your skills and have some fun!


Check it out and see how a simple card can pack a fun punch! 🚀😄
